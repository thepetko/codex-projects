# Dovolenka dashboard

Interaktivny dashboard pre porovnanie archivnych last minute cien z jula 2025 s referencnou dovolenkou z jula 2026.

Hlavna stranka je `index.html`. Subor funguje ako staticka HTML stranka bez externych zavislosti.

Zdrojove data:

- `dovolenka_jul_2025_destination_prices.csv`
- `dovolenka_jul_2025_hotel_candidates.csv`
- `dovolenka_jul_2025_report.md`

Poznamka: Historicke ceny su archivovane ceny `od` z Dovolenka.SME.sk cez Internet Archive. Pri hotelovych kartach sa nepodarilo priamo v archive potvrdit komplet kriterii 4*+, Ultra/All inclusive, hotelova plaz a bazen.
