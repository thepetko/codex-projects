# Dovolenka dashboard

Interaktivny dashboard pre porovnanie archivnych last minute cien z jula 2025 s referencnou dovolenkou z jula 2026.

Hlavna stranka je `index.html`. Subor funguje ako staticka HTML stranka bez externych zavislosti.

Zdrojove data:

- `dovolenka_jul_2025_destination_prices.csv`
- `dovolenka_jul_2025_hotel_candidates.csv`
- `historical_all_inclusive_packages_july_2025.csv`
- `dovolenka_jul_2025_report.md`

Poznamka: Historicke ceny z Dovolenka.SME.sk su archivovane ceny `od` cez Internet Archive. Pri hotelovych kartach sa nepodarilo priamo v archive potvrdit komplet kriterii 4*+, Ultra/All inclusive, hotelova plaz a bazen.

Doplneny AI/UAI prehlad pouziva archivovane baliky z Invia.sk a Fischer.sk s julovym odchodom 2025. Tieto riadky su v dashboarde oddelene od slabsich hotelovych kariet bez overenej stravy.
